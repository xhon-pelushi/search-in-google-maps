## Detailed description

**Copy this:**

Search in Google Maps adds a right-click option so you can look up any selected text on Google Maps instantly.

**How it works**
• Select text on any webpage (an address, place name, or landmark).
• Right-click and choose **Search in Google Maps**.
• A new tab opens with Google Maps and your search—no copy-paste needed.

**Why use it**
• Saves time when you’re reading articles, emails, or docs and want to see a location on the map.
• Works on any site; the context menu appears whenever you have text selected.
• No account or API key required. Uses only the permissions needed for the context menu and opening a tab.

**Technical**
• Built for Chrome with Manifest V3.
• No data is sent to external servers; searches go directly to Google Maps.
• This is an independent project and is not affiliated with or endorsed by Google.

## Images to upload

**Store icon (required)**  
Use the extension’s icon from this project: **`icons/128.png`** (128×128). That’s the simple Google Maps–style pin icon already in the repo.

**Promo tiles (optional)**  
If the store asks for small (440×280) or marquee (1400×560) promo tiles, use a simple graphic with the same icon from **`icons/128.png`** (e.g. icon on a solid or gradient background with the extension name). No generated assets—keep it to the icon you have.

**Screenshots (at least one required)**  
Use the images in **`store-screenshots/`**—they’re already resized and compliant:

| File | Size | Format |
|------|------|--------|
| `store-screenshots/screenshot-1-context-menu.png` | 1280×800 | 24-bit PNG (no alpha) |
| `store-screenshots/screenshot-2-google-maps.png` | 1280×800 | 24-bit PNG (no alpha) |

Chrome Web Store rules: up to 5 screenshots, 1280×800 or 640×400, JPEG or 24-bit PNG (no alpha). These two meet that.
